How to put images in your program or webpage using imgur.com

First, it's important to distguish between putting an image on the webpage and putting an image into the code of the program. If you just want a static, non moving picture, it's probably easier to put the image in the html code of th webpage. If you want to animate the picture or have interact with other objects in the program, you'll probably want to put it in the .pde code file.

Git Hub doesn't let you store large files like images or music. If you want to put images into your program or website, you'll first need a image hosting services. I'm going to use imgur.com. First, upload your images to imgur by clicking on "Computer", selecting the images and clicking start upload.

Once your images are online, you use the "Direct Link (email & IM)" to place the picture on either your webpage or in the program

To put the image on the webpage, I use something like this image html tag <img src="http://i.imgur.com/aAHK0LA.png"> in the body of the index.html. You can copy and paste from the box labeled "HTML Image (websites / blogs)" on imgur.

To put the image in a processing program. I need to declare a PImage variable, initialize it and draw it to the screen with the image() function. Here's a simple program that loads a mug shot of Bill Gates. 

PImage b; //Declare a PImage named b
void setup()
{
	size(450,315);
	b = loadImage("http://imgur.com/8s8eke8.jpg"); //initialize with imgur url
}
void draw()
{
	image(b, 0, 0,450,315); //image has top left corner at (0,0) width of 450 and height of 315
}
